# METEO24 - – Regional Disaster Monitor
A Regional Disaster Management Monitor which extracts data from open weather data resources and displays it onto the web dashboard. Currently , the data is limited upto the state of West Bengal which will be expanded to the rest of the states afterwards. 

Built using **HTML**, **CSS**, **JavaScript**, **SVG**, **Open-Meteo API**, and **USGS Earthquake API**.

# ✨ Key Features

- 🗺️ **Interactive West Bengal Map**
  - Click any district to inspect live conditions
  - Color-coded disaster risk markers
  - Responsive SVG-based visualization

- 🌤️ **Live Weather Monitoring**
  - Real-time Temperature
  - Humidity
  - Wind Speed
  - Current Rainfall
  - Daily Precipitation

- 🚨 **Disaster Risk Assessment**
  - Flood Risk
  - Storm / Tornado Risk
  - Tsunami Risk (Coastal)
  - Earthquake Activity

- 🌍 **Live Seismic Monitoring**
  - Recent earthquakes from USGS
  - Magnitude
  - Location
  - Event timeline

- 💻 **Modern Dashboard**
  - Cyber-inspired dark interface
  - Live clock
  - Live status indicator
  - Responsive design

---

# 📊 Risk Assessment

The dashboard estimates disaster risks using live environmental data.

| **Parameter**        | **Used For** |
|----------------------|--------------|
| Rainfall             | Flood Risk |
| Humidity             | Flood Risk |
| Wind Speed           | Storm Risk |
| Earthquake Magnitude | Seismic Risk |
| Coastal Activity     | Tsunami Risk |
